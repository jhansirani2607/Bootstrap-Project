FresherConnect – Jobs & Internships for Freshers 
About the Project 
FresherConnect is a responsive website that helps freshers find jobs and internships. 
It is built using Bootstrap 5, HTML5, and CSS3. 
Pages 
1. Home Page (index.html) 
• Navbar with links to Home, About, Apply 
• Hero section with a welcome message and Apply button 
• Job Categories using cards with badges 
• Latest job openings using list groups 
• “Why Choose Us” section 
• Footer 
2. About Page (about.html) 
• Intro section about FresherConnect 
• Statistics using grid system 
• Timeline of our journey 
• Values section (Transparency, Accessibility, Growth) 
• Call-to-action to apply 
3. Contact / Apply Page (contact.html) 
• Info cards on the left side 
• Application form on the right side: 
o Full Name, Email, Qualification, Role, Skills 
o About Yourself textarea 
o Resume upload field 
• Offcanvas message for successful submission 
• Footer 
Bootstrap Components Used 
• Navbar – navigation on all pages 
• Cards – job categories, info panels 
• Forms – application form 
• Offcanvas – success message after form submission 
• Grid System – page layout and responsiveness 
How I Built It 
1. Planned the layout and sections for each page 
2. Used Bootstrap grid system for responsive design 
3. Added cards, badges, forms, and offcanvas components 
4. Tested on desktop, tablet, and mobile 
5. Deployed on GitHub Pages 
Challenges & Solutions 
• Aligning form fields on mobile → used responsive grid classes 
• Making images responsive → added .img-fluid class 
• Offcanvas not showing → included Bootstrap JS bundle 
What I Learned 
• Hands-on experience with Bootstrap 5 
• How to design clean and responsive web pages 
• How to remix Bootstrap components creatively
